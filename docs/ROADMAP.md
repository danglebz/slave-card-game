# 🗺️ Roadmap — ไพ่สลาฟออนไลน์

เอกสารรวมไอเดียฟีเจอร์/หน้า/ฟังก์ชันที่อยากเพิ่ม จัดกลุ่มตามคุณค่า (เน้นบริบท **เล่นในออฟฟิศ / LAN**)

> สถานะ: `1.2.0` · อัปเดต: 2026-06-26

---

## ✅ มีแล้ว (baseline)

- 2 หน้า: **ล็อบบี้** (สร้าง/เข้าห้อง) + **โต๊ะเล่น**
- เกม server-authoritative (ตรรกะอยู่ฝั่ง server โกงไม่ได้)
- Persist ห้องลง `rooms.json` + reconnect ที่นั่งเดิมด้วยชื่อ (กัน server restart)
- Grace period เก็บกวาดห้องว่าง (รอ reconnect 10 นาที)
- Modal กติกา + Modal ผลรอบ + Toast แจ้งเตือน
- Form validation (Valibot) + ไอคอน Lucide
- เข้าห้องผ่าน `?room=CODE` + คัดลอกรหัสห้อง
- แสดงเลขเวอร์ชัน (ฝังจาก `package.json` ตอน build)

## 🔥 เฟส 1 — คุ้มสุด เล่นในออฟฟิศต้องมี

| ฟีเจอร์ | รายละเอียด | เหตุผล |
|---|---|---|
| ✅ ⏱️ **Turn timer + auto-pass** | **(เสร็จ v1.3)** นับถอยหลัง 30 วิ/ตา หมดเวลา → มีกอง=ผ่าน, นำกอง=ลงไพ่ต่ำสุด · ปรับเวลาได้ด้วย `TURN_MS` | คนออฟฟิศวอกแวก/เปิดแท็บอื่น → เกมดอง |
| ✅ 🔔 **แจ้งเตือน "ถึงตาคุณ"** | **(เสร็จ)** เสียง (WebAudio) + สั่นมือถือ + แฟลช title แท็บ · เปิด/ปิดในเมนูตั้งค่า (ส่วนตัว) | กันลืมตาเวลาไปดูจออื่น |
| 💬 **แชท + อิโมจิ reactions** | กล่องแชทในห้อง + ปุ่มส่งอิโมจิเร็ว (👍😂🔥💀) ลอยบนจอ | หัวใจเกมสังสรรค์ — ยั่ว/เชียร์กัน |
| 📊 **Scoreboard สะสมข้ามรอบ** | นับสถิติในเซสชัน (ใครเป็นคิง/สลาฟกี่ครั้ง) | มีของให้แข่งทั้งวัน |

## 🎮 เฟส 2 — เพิ่มความลึก / ความสนุก

| ฟีเจอร์ | รายละเอียด |
|---|---|
| ✅ 🤖 **บอท (AI) เติมคน** | **(เสร็จ)** หัวห้องเพิ่ม/ลบบอทในล็อบบี้ · AI เดินเอง (นำ=ทิ้งต่ำสุด, ตาม=ชุดเล็กสุดที่กินได้, แลกไพ่อัตโนมัติ) · ปรับ delay ด้วย `BOT_MS` |
| 🟡 ⚙️ **ตั้งค่าห้องได้** | **(แล้ว: timer/auto-pass + เวลาต่อตา 15/30/45/60 วิ หัวห้องคุม)** เพิ่มต่อ: เปิด/ปิดบอมบ์ · จำนวนรอบจบเกม · จำนวนคนสูงสุด |
| ✅ 👀 **Spectator mode** | **(เสร็จ)** เข้ากลางเกม=ผู้ชม (เห็นโต๊ะ ไม่เห็นมือคนอื่น เล่นไม่ได้) · ถูกดึงเข้าเป็นผู้เล่นรอบถัดไป · แบนเนอร์ + ตัวนับผู้ชมบน topbar |
| 🟡 🎬 **Animation** | **(เริ่ม: ไพ่ลงกองเด้งเข้า + เขย่าตอนบอมบ์, respect reduced-motion)** เพิ่มต่อ: แจกไพ่/เอฟเฟกต์ชนะ |
| ✅ 🔄 **Quick rematch / สลับที่นั่ง** | **(เสร็จ)** "เล่นรอบใหม่" มีอยู่แล้ว (ปุ่ม again) + เพิ่มปุ่ม "สลับที่นั่ง" (หัวห้อง, ล็อบบี้) สุ่มลำดับการวนใหม่ |
| ✅ 📱 **QR code แชร์ห้อง** | **(เสร็จ — ปุ่มซ่อนไว้ก่อน)** modal QR ของลิงก์ `?room=CODE` + ปุ่มคัดลอกลิงก์ ยังอยู่ในโค้ด · ปุ่มเข้า (topbar) ใส่ `hidden` ไว้ เปิดกลับได้ทันที |
| ✅ 🔊 **เสียงเอฟเฟกต์** | **(เสร็จ)** สังเคราะห์สดด้วย WebAudio: ลงไพ่ · บอมบ์ · เคลียร์กอง · ชนะ/แพ้ · toggle ในเมนูตั้งค่า (เปิดเป็นค่าเริ่มต้น) |
| 🔊 **เสียงเอฟเฟกต์** | ลงไพ่ · บอมบ์ · ชนะ · ถึงตา (เปิด/ปิดได้) |
