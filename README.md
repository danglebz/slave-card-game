# 🃏 ไพ่สลาฟ — เกมไพ่ออฟฟิศ

เกมไพ่สลาฟออนไลน์ เล่นพร้อมกันหลายเครื่อง 2–4 คน ผ่านเบราว์เซอร์ (มือถือ/คอม)
ตรรกะเกมอยู่ฝั่งเซิร์ฟเวอร์ทั้งหมด (server-authoritative) โกงไม่ได้

## เริ่มเล่น

```bash
pnpm install      # ติดตั้งครั้งแรก
pnpm start        # รันเซิร์ฟเวอร์
```

จากนั้นเปิดลิงก์ที่ขึ้นในเทอร์มินัล เช่น:

```
   เครื่องนี้:   http://localhost:3000
   ในออฟฟิศ:   http://192.168.1.50:3000   ← ส่งลิงก์นี้ให้เพื่อน
```

เพื่อนในออฟฟิศ (Wi-Fi เดียวกัน) เปิดลิงก์ `http://<ไอพีเครื่องคุณ>:3000` ได้เลย ไม่ต้องลงอะไร

> เปลี่ยนพอร์ตได้ด้วย `PORT=8080 pnpm start`

## วิธีเล่น

1. คนแรกกด **สร้างห้องใหม่** → ได้รหัสห้อง 4 ตัว (เช่น `AB12`)
2. คนอื่นใส่ชื่อ + รหัสห้อง กด **เข้าร่วมห้อง** (หรือเปิดลิงก์ที่แชร์ได้เลย รหัสจะเด้งใส่ให้)
3. หัวห้อง (👑) กด **เริ่มเกม** เมื่อครบคน

## กติกา (มาตรฐาน)

- อันดับไพ่: `3` ต่ำสุด → `4 5 6 7 8 9 10 J Q K A` → `2` สูงสุด
- ดอกตัดสินเมื่ออันดับเท่ากัน: ♣ < ♦ < ♥ < ♠
- คนถือ **3♣** ขึ้นก่อน และไพ่กองแรกของเกมต้องมี 3♣ ร่วมด้วย
- ลงได้: **เดี่ยว / คู่ / ตอง / เรียง (≥3 ใบ ห้ามมี 2) / โฟร์ (ระเบิด)**
- ต้องลงชุดประเภทเดียวกันและสูงกว่ากองบนโต๊ะ (เรียงต้องยาวเท่ากัน)
- **โฟร์ (ระเบิด)** กินได้ทุกอย่าง (ระเบิดใหญ่กินระเบิดเล็ก)
- ลงไม่ได้/ไม่อยากลงก็ **ผ่าน**; ถ้าทุกคนผ่านหมด เจ้าของกองล่าสุดได้นำกองใหม่
- ใครหมดมือก่อนเป็น **เศรษฐี** 🏆 คนสุดท้ายเป็น **สลาฟ** 💀
- หัวห้องกด **เล่นรอบใหม่** เพื่อแจกไพ่ใหม่

## โครงสร้าง

```
server/
  game.js   ตรรกะไพ่ล้วน (สร้าง/แจก/ระบุชุด/เทียบกอง) — ทดสอบง่าย
  room.js   สถานะห้อง + รอบเล่น (state machine)
  index.js  Express + Socket.IO
public/
  index.html / style.css / app.js   ฝั่งเบราว์เซอร์ (vanilla JS)
```
